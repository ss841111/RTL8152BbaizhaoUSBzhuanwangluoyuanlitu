# RTL8152B 百兆USB转网络原理图

## 资源描述

本仓库提供了一份关于RTL8152B百兆USB转网络的原理图资源文件。RTL8152B是台湾瑞昱(REALTECK)公司推出的一款USB2.0转10/100M网络芯片，它由符合IEEE802.3u标准的MAC、USB控制器、内存、低压差线性稳压器等组成。该芯片主要应用于USB10/100M以太网主板、电子狗、笔记本、扩充口、嵌入式系统等领域。
